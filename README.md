# Bank Project (Array Based)

This is a Java console-based bank user application.
The project is implemented using arrays and core Java concepts.

## Features
- Create bank users
- Validate user input
- Handle exceptions
- Perform basic banking operations

## Technologies Used
- Java
- Core Java
- Eclipse IDE

## How to Run
1. Clone the repository
2. Open the project in Eclipse
3. Run the UserAdmin.java file
